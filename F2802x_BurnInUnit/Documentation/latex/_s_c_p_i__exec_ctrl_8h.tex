\subsection{Detailed Description}
T\-O\-D\-O\-: Currently the calling function should call operation\-Complete() when \hyperlink{_s_c_p_i__exec_ctrl_8h_addbde078f73d385a37fbb92fb3054e4d}{exec\-Ctrl()} has finished all commands.

The execution control block deals with coupled parameters and queries that require a device action before they can generate a $ \langle$ R\-E\-S\-P\-O\-N\-S\-E M\-E\-S\-S\-A\-G\-E $ \rangle$ . The execution control determines when the device has gathered enough information to execute a command. When the execution control has gathered enough information to initiate a device action, it sends an executable message element to the device functions. The execution control will wait for the resulting device actions to complete (sequential command). A $ \langle$ P\-R\-O\-G\-R\-M M\-E\-S\-S\-A\-G\-E U\-N\-I\-T $ \rangle$ is considered executed when it has been parsed and all corresponding device operations have been either completed or initiated.

This file should not need to be edited by the user.
