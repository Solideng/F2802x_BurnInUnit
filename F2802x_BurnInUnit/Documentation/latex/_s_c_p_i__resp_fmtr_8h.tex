\subsection{Detailed Description}
This file provides functions and definitions that form the S\-C\-P\-I response formatter.

The response formatter builds a $ \langle$ R\-E\-S\-P\-O\-N\-S\-E M\-E\-S\-S\-A\-G\-E $ \rangle$ out of response message elements from valid query requests and response data. The $ \langle$ R\-E\-S\-P\-O\-N\-S\-E M\-E\-S\-S\-A\-G\-E $ \rangle$ is placed in to the output queue. The response formatter's primary responsibility is to convert the internal representation of data elements into a sequence of data bytes according to the I\-E\-E\-E 488.\-2 Device Talking Elements syntax rules.

This file should not need to be edited by the user.
